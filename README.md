[オラクル社から、弊社名の登録商標「Javaspsarrow」に「JAVA」が含まれるため、商標権の一部について権利放棄を可否を問う連絡を受けました。現在、本件検討のための回答期間として9月30日までの猶予を希望する旨をオラクル社へ連絡したところです。 - Twitter 2019/8/30](https://twitter.com/javasparrowtyo/status/1167253699493679104)

[本商標についてオラクル社が特許庁へ異議申し立てを行いましたが、特許庁からは登録は取り消しにならない「維持決定」との判断が下っています。これを受けてオラクル社からは商標登録を無効とする審判の請求を検討しており、裁判となる前に商標の権利を放棄できるかどうかを問われている状態です。](https://twitter.com/javasparrowtyo/status/1167253813788459009)

[ニュースリリース 「オラクル社より弊社商標「Javasparrow」の商標登録に対する無効審判請求を前提とした商標権一部放棄の検討要求を受けました 」](https://twitter.com/javasparrowtyo/status/1167253954029187073)

[A: 商標権を放棄するにあたり、オラクル社とは金銭的な交渉はまだ一切しておらず、まずはシンプルに商標権一部を放棄してほしい、ということだけ通達を受けています。](https://twitter.com/javasparrowtyo/status/1174881747109994496)

[弊社登録商標「Javasparrow」について。弊社はオラクル社からの要望に対して、「Javasparrow」の商標取得に掛かった費用や放棄するにあたり要する費用など、金銭的な交渉を条件とした放棄の意向を有すると、9月26日付けの書面にて回答しました。なお現時点、具体的な交渉等は始まっていません。 - 2019/10/16](https://twitter.com/javasparrowtyo/status/1184374252583014401)
